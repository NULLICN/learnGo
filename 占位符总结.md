# Go 语言占位符（Format Verbs）详解

## 概述
Go 语言中的占位符用于在格式化输出函数（如 `fmt.Printf`）中指定值的输出格式。最常见的函数有：
- `fmt.Printf(format, args...)` - 格式化输出
- `fmt.Sprintf(format, args...)` - 格式化字符串
- `fmt.Fprintf(w, format, args...)` - 格式化写入

---

## 占位符分类

### 1️⃣ 通用占位符

| 占位符 | 说明 | 例子 |
|------|------|------|
| `%v` | 默认格式表示值 | `fmt.Printf("%v", 42)` → `42` |
| `%#v` | Go 语法格式表示 | `fmt.Printf("%#v", []int{1,2})` → `[]int{1, 2}` |
| `%T` | 值的类型 | `fmt.Printf("%T", 42)` → `int` |
| `%%` | 百分号 | `fmt.Printf("%%")` → `%` |

**使用场景：**
```go
x := 42
fmt.Printf("%v\n", x)    // 42
fmt.Printf("%#v\n", x)   // 42
fmt.Printf("%T\n", x)    // int
```

---

### 2️⃣ 布尔占位符

| 占位符 | 说明 | 例子 |
|------|------|------|
| `%t` | 布尔值（true/false） | `fmt.Printf("%t", true)` → `true` |

**使用场景：**
```go
flag := true
fmt.Printf("%t\n", flag)   // true
fmt.Printf("%t\n", 1 > 0)  // true
```

---

### 3️⃣ 整数占位符

| 占位符 | 说明 | 例子 |
|------|------|------|
| `%d` | 十进制 | `fmt.Printf("%d", 255)` → `255` |
| `%b` | 二进制 | `fmt.Printf("%b", 255)` → `11111111` |
| `%o` | 八进制 | `fmt.Printf("%o", 255)` → `377` |
| `%x` | 十六进制（小写） | `fmt.Printf("%x", 255)` → `ff` |
| `%X` | 十六进制（大写） | `fmt.Printf("%X", 255)` → `FF` |
| `%c` | Unicode 字符 | `fmt.Printf("%c", 65)` → `A` |
| `%q` | 转义后的字符 | `fmt.Printf("%q", 65)` → `'A'` |

**使用场景：**
```go
num := 255
fmt.Printf("%d\n", num)  // 255（十进制）
fmt.Printf("%x\n", num)  // ff（十六进制）
fmt.Printf("%b\n", num)  // 11111111（二进制）
fmt.Printf("%c\n", 65)   // A（对应字符）
```

---

### 4️⃣ 浮点数占位符

| 占位符 | 说明 | 例子 |
|------|------|------|
| `%f` | 小数点表示 | `fmt.Printf("%f", 3.14)` → `3.140000` |
| `%e` | 科学记数法（小写 e） | `fmt.Printf("%e", 3.14)` → `3.140000e+00` |
| `%E` | 科学记数法（大写 E） | `fmt.Printf("%E", 3.14)` → `3.140000E+00` |
| `%g` | 根据大小自动选择 | `fmt.Printf("%g", 3.14)` → `3.14` |
| `%G` | 根据大小自动选择（大写） | `fmt.Printf("%G", 3.14)` → `3.14` |

**使用场景：**
```go
pi := 3.14159
fmt.Printf("%f\n", pi)   // 3.141590
fmt.Printf("%.2f\n", pi) // 3.14（精度为2）
fmt.Printf("%e\n", pi)   // 3.141590e+00
fmt.Printf("%g\n", pi)   // 3.14159
```

---

### 5️⃣ 字符串占位符

| 占位符 | 说明 | 例子 |
|------|------|------|
| `%s` | 字符串 | `fmt.Printf("%s", "Hello")` → `Hello` |
| `%q` | 带双引号的字符串 | `fmt.Printf("%q", "Hello")` → `"Hello"` |
| `%x` | 十六进制（小写） | `fmt.Printf("%x", "Hi")` → `4869` |
| `%X` | 十六进制（大写） | `fmt.Printf("%X", "Hi")` → `4869` |

**使用场景：**
```go
str := "Hello"
fmt.Printf("%s\n", str)   // Hello
fmt.Printf("%q\n", str)   // "Hello"
fmt.Printf("%x\n", str)   // 48656c6c6f（十六进制编码）
```

---

### 6️⃣ 字节占位符

| 占位符 | 说明 | 例子 |
|------|------|------|
| `%s` | 字节切片 | `fmt.Printf("%s", []byte{72,101})` → `He` |
| `%q` | 转义的字节切片 | `fmt.Printf("%q", []byte{72,101})` → `"He"` |
| `%x` | 十六进制 | `fmt.Printf("%x", []byte{72,101})` → `4869` |

**使用场景：**
```go
bytes := []byte{72, 101, 108, 108, 111}  // "Hello"
fmt.Printf("%s\n", bytes)   // Hello
fmt.Printf("%x\n", bytes)   // 48656c6c6f
```

---

### 7️⃣ 指针占位符

| 占位符 | 说明 | 例子 |
|------|------|------|
| `%p` | 十六进制指针地址 | `fmt.Printf("%p", &x)` → `0xc000010080` |

**使用场景：**
```go
x := 42
ptr := &x
fmt.Printf("%p\n", ptr)  // 0xc000010080（内存地址）
fmt.Printf("%v\n", ptr)  // 0xc000010080（同上）
```

## 修饰符

### 🔧 宽度修饰符

```go
// 右对齐，宽度为5
fmt.Printf("%5d\n", 42)    // "   42"

// 左对齐，宽度为5
fmt.Printf("%-5d\n", 42)   // "42   "

// 零填充，宽度为5
fmt.Printf("%05d\n", 42)   // "00042"

// 字符串精度（最多3个字符）
fmt.Printf("%.3s\n", "Hello")  // "Hel"
```

### 🔧 精度修饰符

```go
pi := 3.14159

// 小数点后2位
fmt.Printf("%.2f\n", pi)    // "3.14"

// 宽度5，小数点后2位
fmt.Printf("%5.2f\n", pi)   // " 3.14"
```

### 🔧 标志符

| 标志 | 说明 | 例子 |
|------|------|------|
| `+` | 总是显示符号 | `fmt.Printf("%+d", 42)` → `+42` |
| `-` | 左对齐 | `fmt.Printf("%-5d", 42)` → `42   ` |
| `0` | 零填充 | `fmt.Printf("%05d", 42)` → `00042` |
| ` ` | 空格表示正数 | `fmt.Printf("% d", 42)` → ` 42` |
| `#` | 显示进制前缀 | `fmt.Printf("%#x", 255)` → `0xff` |

**使用场景：**
```go
fmt.Printf("%+d\n", 42)      // "+42"（带符号）
fmt.Printf("% d\n", 42)      // " 42"（正数前空格）
fmt.Printf("%#x\n", 255)     // "0xff"（显示16进制前缀）
fmt.Printf("%#o\n", 64)      // "0100"（显示8进制前缀）
```

## 实际应用示例

### 示例 1：格式化用户信息
```go
type User struct {
    ID   int
    Name string
    Age  float32
}

user := User{ID: 1, Name: "张三", Age: 28.5}

fmt.Printf("用户ID: %d\n", user.ID)
fmt.Printf("用户名: %s\n", user.Name)
fmt.Printf("年龄: %.1f\n", user.Age)
fmt.Printf("完整信息: %#v\n", user)
fmt.Printf("类型: %T\n", user)
```

### 示例 2：格式化表格
```go
fmt.Printf("%-10s %10d %10.2f\n", "商品", "数量", "价格")
fmt.Printf("%-10s %10d %10.2f\n", "苹果", 5, 3.50)
fmt.Printf("%-10s %10d %10.2f\n", "香蕉", 10, 2.99)
```

### 示例 3：日志输出
```go
x := 42
fmt.Printf("[DEBUG] x = %v (type: %T)\n", x, x)
fmt.Printf("[INFO] Memory address: %p\n", &x)
fmt.Printf("[ERROR] Invalid: %#v\n", []int{})
```

## 常用组合总结

| 用途 | 占位符组合 | 说明 |
|------|---------|------|
| 十进制整数 | `%d` | 基本整数输出 |
| 浮点数（2位小数） | `%.2f` | 金额、百分比等 |
| 字符串 | `%s` | 基本字符串 |
| 带引号字符串 | `%q` | 用于日志或调试 |
| 调试输出 | `%#v` | 查看完整结构体 |
| 类型信息 | `%T` | 调试类型问题 |
| 内存地址 | `%p` | 指针调试 |
| 默认格式 | `%v` | 通用输出 |

---

## 总结

- **常用占位符**：`%v`、`%d`、`%s`、`%f`
- **调试占位符**：`%#v`、`%T`、`%p`
- **进制转换**：`%b`（二进制）、`%o`（八进制）、`%x/%X`（十六进制）
- **修饰符**：宽度、精度、标志符可灵活组合
- **最佳实践**：使用 `%#v` 查看完整结构体，使用 `%T` 检查类型
